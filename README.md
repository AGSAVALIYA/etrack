## Run Locally

Clone the project

```bash
  git clone https://github.com/AGSAVALIYA/etrack
```

Go to the project directory

```bash
  cd etrack
```
Create your env file with firebase config.

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
